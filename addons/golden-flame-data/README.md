# IGF + Suldan

## Installation

```js
npm add @massif/osr-data
//or
yarn add @massif/osr-data
```

## Usage

```js
import * as osr from '@massif/osr-data';

const osrFrameData = osr.frames;
```

## Building LCP

```js
yarn && yarn build
// or
npm install && npm run build
```

produces a versioned LCP file in `dist/`
